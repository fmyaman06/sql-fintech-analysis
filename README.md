# 🏦 FinTech SQL & Veri Analizi

Finansal işlem verilerini SQL ve Python ile analiz eden portfolyo projesi.

## 📊 İçerik

| Notebook | Konu | Araçlar |
|----------|------|---------|
| hafta1_sql_eda.ipynb | Kategori & durum bazında işlem analizi | SQL, pandas, matplotlib |

## 🛠️ Kullanılan Teknolojiler
- **PostgreSQL** — veri tabanı
- **Python** — pandas, matplotlib, sqlalchemy
- **Jupyter Notebook** — analiz ortamı

## 📈 Analizler
- Kategori bazında toplam ve ortalama işlem tutarı
- Durum bazında işlem dağılımı (onaylı, iptal, beklemede)
- En yüksek tutarlı işlemler
